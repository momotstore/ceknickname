Overview

Motmot Store API memungkinkan Anda mengambil informasi pemain dari game mobile populer termasuk Free Fire dan Mobile Legends. API ini mengembalikan detail pemain seperti username, user ID, dan informasi zona (server).

Base URL
https://xstbot-store.vercel.app/

Endpoints

GET

Free Fire Checker

/api/game/free-fire ? id = 123456789
Parameter Type Required Description
id String / Number YES ID Player Free Fire yang valid
GET

Mobile Legends Checker

/api/game/mobile-legends ? id = 123456789 & zone = 1234
Parameter Type Required Description
id String / Number YES ID Player Mobile Legends
zone String / Number YES Zone ID (Server ID)

Response Format

Success Response 
{
  "code": 200,
  "status": "true",
  "message": "ID berhasil ditemukan",
  "data": {
    "username": "Motmot Store",
    "user_id": "123456789",
    "zone": "1234" 
  }
}
Failed Response 
{
  "code": 404,
  "status": "false",
  "message": "ID tidak ditemukan"
}

Status Codes

Code Status Description
200 Success Request berhasil dan data ditemukan.
404 Not Found ID Player tidak ditemukan atau salah input.
500 Server Error Terjadi kesalahan pada server internal.
